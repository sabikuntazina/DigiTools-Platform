# Project Name: DigitalTools

## Description

A modern and responsive web application where users can explore digital products, add them to a cart, and purchases easily with a smooth and interactive UI.


## Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* React-Toastify
* JSON


## Features:

### 1. Smart Cart System

Add, remove products and view total items dynamically in the navbar.

### 2. Toggle View

Easily switch between Products and Cart sections with interactive buttons.

### 3. Proceed to checkout

For Payment done, added total cost and Proceed to checkout button.

### 3. Toast Notifications

Real-time alerts for add to cart, remove items, and checkout actions.
